Volume Segmantics
===============================

This documentation site for the Vol-Seg PyTorch Deep-learning Segmentation Toolkit is compliled into 3 main sections; 

- *The Main ReadMe* document; containing the basic instructions and information needed to get started with the pakcages usage.
- *The Advanced Usage and Functionality* document; containing more detailed installation instructions for non-coder usage alongside developmental users, directions for train/predict setting .yaml file editing and fine-tuning, segmentation workflow guidlines, guidance for data visualisation and editing and explainations for post-processing procedures and protocols used in conjunction with additional toolkits for efficient segmention.
- *A Walk-through Tutorial*; highlighting the toolkits main uasage capabilities using high-res open source medical data (WINSdata).

First-time users with minimal codeing experience are advised to read through main (ReadMe) and detailed installation instructions (Usage and Functionality) before then experimenting with the 'Walkthrough tutorial' and its data. Users with more codeing experience are also advised to make full use of the Advanced Usage and Functionality guidence to maximise the installations full utility.

The files for this documentation can also be found installed with the Vol-Seg package alongside the scripts and notebooks mentioned.
